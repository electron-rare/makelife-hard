# Design Rules

Guidelines and constraints for reusable design blocks.

## Power

- Input voltage range and derating
- Decoupling capacitor placement
- Thermal pad requirements
- Load transient considerations

## I2C

- Pull-up resistor sizing for bus speed
- Address allocation and conflict avoidance
- Bus capacitance budget
- Level shifting requirements

## Battery

- Cell balancing topology
- Charge/discharge current limits
- Protection IC coordination
- Thermal management zones

## Audio

- Signal-to-noise ratio targets
- Ground plane separation (analog/digital)
- Decoupling and filtering
- I2S routing constraints

## LED

- Current limiting resistor tolerances
- PWM frequency selection
- Thermal derating curves
- Data line termination (addressable LEDs)

## Motor

- Flyback diode selection
- Current sense resistor placement
- EMI filtering requirements
- Gate drive isolation
